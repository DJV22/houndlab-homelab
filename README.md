# HoundLab Homelab

Personal homelab built on Proxmox VE using a MinisForum UM890 PRO.
Focused on self-hosted, open-source services for personal use.

## Hardware
- MinisForum UM890 PRO
- 96GB RAM
- 2x4TB NVMe (ZFS mirror)
- External 6-bay USB enclosure
- 3x20TB Seagate IronWolf Pro
